|-------|-------------|

| [MemberSim Examples](membersim-examples.md) | Claims data: professional, facility, adjudication |

## How to Use These Examples

Each example shows:
1. **The Prompt** - What to ask Claude
2. **Expected Output** - What you should receive
3. **Key Points** - What to notice in the output
4. **Variations** - How to modify the request
