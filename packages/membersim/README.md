# MemberSim

Synthetic insurance member data generation for payer/claims scenarios.

## Overview

MemberSim is part of the HealthSim workspace, providing realistic synthetic member data generation for healthcare insurance simulation scenarios.

## Features

- Member enrollment and eligibility simulation
- Professional and facility claims generation
- X12 837/835/834 output formats
- Value-based care scenarios

## Installation

```bash
pip install -e .
```
