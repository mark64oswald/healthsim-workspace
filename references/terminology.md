# Healthcare Terminology Reference

## Overview

This reference provides common healthcare terminology, abbreviations, and acronyms used across HealthSim data generation. Use this for consistent naming and proper abbreviation expansion.

## Clinical Abbreviations

### Vital Signs

| Abbreviation | Full Term | Context |
|--------------|-----------|---------|
| BP | Blood Pressure | Measured in mmHg (e.g., 120/80) |
| HR | Heart Rate | Beats per minute (bpm) |
| RR | Respiratory Rate | Breaths per minute |
| SpO2 | Oxygen Saturation | Percentage (pulse oximetry) |
| Temp | Temperature | Fahrenheit or Celsius |
| Wt | Weight | Pounds (lbs) or kilograms (kg) |
| Ht | Height | Inches or centimeters |
| BMI | Body Mass Index | kg/m² |
